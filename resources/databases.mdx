# Databases and libraries

## Relational Database

- [MySQL](https://github.com/mysqljs/mysql) - Sql Database
- [PostgreSQL](https://www.postgresql.org/) - The World's Most Advanced Open Source Relational Database
- [MariaDB](https://www.postgresql.org/) - MariaDB Server: The open source relational database
- [AWS RDS](https://aws.amazon.com/rds/) - Amazon Relational Database Service (RDS)
- [Sequelize](https://sequelize.org/) - Sequelize is a promise-based Node.js ORM for Postgres, MySQL, MariaDB, SQLite and Microsoft SQL Server
- [Oracle DB](https://www.oracle.com/database/) - Oracle database products offer customers cost-optimized and high-performance versions of Oracle Database, the world's leading converged, multi-model database management system, as well as in-memory, NoSQL and MySQL databases
- [SQLite](https://www.sqlite.org/index.html) - SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine
- [Bookself JS](https://bookshelfjs.org/) - Bookshelf is a JavaScript ORM for Node.js, built on the Knex SQL query builder
- [MS Sql Server](https://www.microsoft.com/en-ca/sql-server/sql-server-2016) - Sql platform built by Microsoft
- [DB2](https://www.ibm.com/in-en/analytics/db2) - Transform and modernize your data management with AI-driven solutions
- [InterBase](https://bit.ly/3dtlKRK) - InterBase is a full-featured, high-performance, scalable, lightweight, embeddable and encryptable relational database

## Graph Database

- [Neo4j](https://neo4j.com/) - Neo4j is an open-source NoSQL graph database, implemented in Java
- [OrientDB](https://orientdb.org/) - OrientDB is an open source NoSQL database management system written in Java.
- [Dgragh](https://dgraph.io/) - Dgraph is the simplest way to implement a GraphQL backend for your applications
- [Amazon Neptune](https://aws.amazon.com/neptune/) - Build and run graph applications with highly connected datasets
- [Datastax](https://www.datastax.com/) - The Open Stack for Modern Data Apps

- [Apache Giraph](https://giraph.apache.org/) - Apache Giraph is an iterative graph processing system built for high scalability
- [ArchiGraph](https://archigraph.pro/) - An ontology-based platform for data-centric architectures
